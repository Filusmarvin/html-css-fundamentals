Previous versions of HTML offered very few elements for grouping content together. HTML5 has much richer options, so use them!

Sectioning and Semantics

This page illustrates some of the new elements available for grouping content in HTML5, in particular header, section, aside, and footer.

What Sectioning Elements Look Like

By default, the sectioning elements don't have any special styles applied to them, so the end user can't tell they're being used. These elements are to communicate to other developers (and, eventually, to computers parsing sites).

This has been a brief introduction to sectioning in HTML5.
